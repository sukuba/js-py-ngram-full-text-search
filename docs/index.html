js-py-ngram-full-text-search is a project of JsNgram that is a simple N-gram full text search library on JavaScript.

  • Pure JavaScript with jQuery.
  • OS independent.
  • Works on many browsers including mobiles.
  • No http servers required.
  • No database softwares required.
  • Portable.
  • Python backend package as an index builder.
  • Designed for unicode characters like Japanese Kanji.

This document is published at https://sukuba.github.io/js-py-ngram-full-text-search/.
