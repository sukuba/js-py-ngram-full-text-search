JsNgram.add_files_to_json

../.. Reference .. jsngram.jsngram . JsNgram class

method of class JsNgram

Description

Add to index by a list of files and append to json index files using jsngram.json2.json_append function.

Example 
self.add_files_to_json(['a/b.txt', 'a/c.txt'], True)
Usage

add_files_to_json(paths, verbose)

Argument paths is a list of relative paths of the files. Relative to the property src. And each path will be an id of each document. Prints debugging informations if verbose = True.
