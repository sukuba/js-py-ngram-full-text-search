JsNgram.dest

../.. Reference .. jsngram.jsngram . JsNgram class

property of class JsNgram

Description

Destination folder that json index files will be written.

Example 
self.dest = '/scratch/a/pub/idx'
Default value 
'.'
Usage 
import jsngram.jsngram

# gives at class initializing
ix = jsngram.jsngram.JsNgram(dest='/scratch/a/pub/idx')

# sets property value later
ix.dest = '/scratch/b/pub/idx'
