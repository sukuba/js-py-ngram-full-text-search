JsNgram.add_file

../.. Reference .. jsngram.jsngram . JsNgram class

method of class JsNgram

Description

Add to index by a file.

Example 
self.add_file('a/b.txt', True)
Usage

add_file(path, verbose)

Argument path is a relative path of the file. Relative to the property src. And the path will be an id of the document. Prints debugging informations if verbose = True.
