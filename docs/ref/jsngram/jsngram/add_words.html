JsNgram.add_words

../.. Reference .. jsngram.jsngram . JsNgram class

method of class JsNgram

Description

Add to index by a document id, word and position. The word is a sequence of text that do not have any delimiters defined by the property ignore. This function uses the property shorter to control key generation.

Example 
self.add_words('a/b.txt', 'cat', 48)
Usage

add_words(path, content, start)

Argument path is a document id, usually a document path. Argument content is a word. Argument start is a position at the start of the content in the whole text.
