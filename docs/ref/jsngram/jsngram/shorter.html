JsNgram.shorter

../.. Reference .. jsngram.jsngram . JsNgram class

property of class JsNgram

Description

Generates indexes just as the size of n if False, n and lower sizes if True.

Example 
self.shorter = False
Default value 
True
Usage 
import jsngram.jsngram

# gives at class initializing
ix = jsngram.jsngram.JsNgram(shorter=False)

# sets property value later
ix.shorter = True
