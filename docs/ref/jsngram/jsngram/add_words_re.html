JsNgram.add_words_re

../.. Reference .. jsngram.jsngram . JsNgram class

method of class JsNgram

Description

Called by add_words. This method is not intended for users to call directly. Uses the 1st argument n for N-gram size instead of using the property n. This method is needed to enable shorter.

Example 
self.add_words_re(1, 'a/b.txt', 'cat', 48)
Usage

add_words_re(n, path, content, start)

Argument n is a size of N-gram. Argument path is a document id, usually a document path. Argument content is a word. Argument start is a position at the start of the content in the whole text.
