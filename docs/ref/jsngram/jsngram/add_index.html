JsNgram.add_index

../.. Reference .. jsngram.jsngram . JsNgram class

method of class JsNgram

Description

Add an index element to the property db.

Example 
self.add_index('ca', 'a/b.txt', 48)
Usage

add_index(key, path, start)

Argument key is the N-gram key string. Argument path is a document id, usually a document path. Argument start is a position at the key found in the whole text.
