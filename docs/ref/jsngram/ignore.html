JsNgram.ignore

.. Reference . JsNgram class

property of class JsNgram

Description

Define the word delimiters those are excluded from indexes.

Example 
self.ignore = r'\s+'
Default value 
r'[\s,.，．、。]+'

Spaces, tabs, line feeds, punctuation marks both in English and Japanese.

Usage 
import jsngram.jsngram

# gives at class initializing
ix = jsngram.jsngram.JsNgram(ignore=r'[\s!-.]+')

# sets property value later
ix.ignore = r'[\s!-.]+'
  • The value must be a valid regular expression pattern string.
  • Non string values like None are prohibited.

more examples

Including periods and spaces to the index 
r'[\f\n\r\t\v]+'

Form feed ,line feed, carriage return, horizontal and vertical tabs.

Including everything into the index 
r'[^\s\S]+'

Technically, this regular expression matches none.
