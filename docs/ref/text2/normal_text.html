jsngram.text2.normal_text

../.. Reference .. jsngram.text2

function in jsngram.text2 module

Description

Normalize to Hanaku alphanum and Zenkaku kana.

Example 
print(jsngram.text2.normal_text(u'aｂ1２ｶキ'))
# u'ab12カキ'
Usage

out_string = jsngram.text2.normal_text(text)

The out_string is a Unicode normalized string of the argument text. The argument text must be a unicode string.
