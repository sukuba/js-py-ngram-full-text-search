jsngram.json2.has_end

../.. Reference .. jsngram.json2

function in jsngram.json2 module

Description

True if the json file has an end bracket of array, namely, ]. It does not perform entire grammer check, but just watch if the end line is the end bracket only. So, it's not a general tool, and it works only with files generated using function json_end in this module.

Example 
print(jsngram.json2.has_end('/scratch/a.json'))
# True or False
Usage

jsngram.json2.has_end(file_name)

Argument file_name must be a full path to a json file.
