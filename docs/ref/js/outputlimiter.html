JsNgram.outputLimiter

.. Reference . JsNgram static class

property of static class JsNgram

Description

The number of hits and documents shown at once in the result.

Example 
JsNgram.outputLimiter = 40;
Default value 
100
Usage

This is a constant number for the system. Users may change this to increase or decrease the number of items shown at once.
