JsNgram.textBase

.. Reference . JsNgram static class

property of static class JsNgram

Description

Specify a directory of text files.

Example 
JsNgram.textBase = 'a/txt/';
Default value 
'txt/'
Usage

This is a constant string for the system. Users must specify a valid relative path to the text files.
