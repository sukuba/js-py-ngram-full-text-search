JsNgram.resultCount

.. Reference . JsNgram static class

property of static class JsNgram

Description

A success message of search.

Example 
JsNgram.resultCount = '%%か所が %%件の文書に見つかりました。';
Default value 
'%% hits in %% documents.'
Usage

This is a constant string for the system. Users may change this to customize the message, mainly for localization. The sequence of %% in the string will be replaced to the number of hits and documents, respectively.

.. Reference . JsNgram static class sources are available at github.
