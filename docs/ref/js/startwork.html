JsNgram.startWork

.. Reference . JsNgram static class

method of static class JsNgram

Description

Initialize the property work with the keyword to search. Then, start searching by kicking the method generateDeferred.

Example 
JsNgram.startWork('cat');
Usage

startWork(what);

Argument what is a query string to search.
