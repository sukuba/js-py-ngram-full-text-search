JsNgram.sprintf

.. Reference . JsNgram static class

method of static class JsNgram

Description

Insert data of array into string format at %%.

Example 
JsNgram.sprintf('%% hits in %% documents.', [101, 17]);
// '101 hits in 17 documents.'
Usage

sprintf(format, arr);

Argument format is string that contains '%%'s to replace with each elment of array. Argument arr is an array that contains data to be inserted.
