JsNgram.escapeHtml

.. Reference . JsNgram static class

method of static class JsNgram

Description

Escape html special characters.

Example 
JsNgram.escapeHtml('<a>');
// returns &lt;a&gt;
Usage

escapeHtml(text);

Argument text is a text unescaped.

It uses following rules: 

& → &amp;
< → &lt;
> → &gt;
" → &quot;
' → &#x27;
` → &#x60;
