JsNgram.keyExt

.. Reference . JsNgram static class

property of static class JsNgram

Description

The file extention string of json index files including the dot.

Example 
JsNgram.keyExt = '.index';
Default value 
'.json'
Usage

This is a constant string for the system. Users may change this to fit the file name from the backend system.
