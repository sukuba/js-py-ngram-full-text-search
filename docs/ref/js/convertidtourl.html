JsNgram.convertIdToUrl

.. Reference . JsNgram static class

method of static class JsNgram

Description

Convert document id to URL.

Example 
JsNgram.convertIdToUrl('a/b c.txt');
// maybe returns idx/a/b%20c.txt
Usage

convertIdToUrl(docId);

Argument docId is a document id string used in json index files.
