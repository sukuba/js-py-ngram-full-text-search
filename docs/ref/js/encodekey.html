JsNgram.encodeKey

.. Reference . JsNgram static class

method of static class JsNgram

Description

Convert text into unicode hex byte pattern string.

Example 
JsNgram.encodeKey('ca');
// returns '00/63/00/61'
Usage

encodeKey(text);

Argument text is a N-gram partial keyword to search. This is used to generate a file name of json from a keyword to search.
