JsNgram.resultSelector

.. Reference . JsNgram static class

property of static class JsNgram

Description

Specify a html element to show the result on search.

Example 
JsNgram.resultSelector = $('#result');
Default value 
undefined
Usage

This is a constant object for the system. Users must specify a valid element.
