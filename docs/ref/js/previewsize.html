JsNgram.previewSize

.. Reference . JsNgram static class

property of static class JsNgram

Description

The preview text length shown in the result.

Example 
JsNgram.previewSize = 400;
Default value 
240
Usage

This is a constant number for the system. Users may change this to increase or decrease the number of text characters shown as preview.
