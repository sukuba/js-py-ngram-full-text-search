JsNgram.indexFileName

.. Reference . JsNgram static class

method of static class JsNgram

Description

Get json file name of index to find text.

Example 
JsNgram.indexFileName('ca');
// maybe returns 'idx/00/63/00/61.json'
Usage

indexFileName(text);

Argument text is a N-gram partial keyword to search. This generates a file name of json from a keyword to search.
