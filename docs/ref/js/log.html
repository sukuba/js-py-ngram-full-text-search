JsNgram.log

.. Reference . JsNgram static class

method of static class JsNgram

Description

Custom log function with verbosity.

Example 
JsNgram.log.v1('level 1 debugging message: ', x);
Usage

log.v0(obj1, obj2,,, ) log.v1(obj1, obj2,,, ) log.v2(obj1, obj2,,, ) log.v3(obj1, obj2,,, )

When verbosity = 1, log.v0 and log.v1 act as console.log, and log.v2 and log.v3 do nothing. The log.v0 is as same as console.log.
