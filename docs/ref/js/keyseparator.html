JsNgram.keySeparator

.. Reference . JsNgram static class

property of static class JsNgram

Description

The directory separator character of json index files.

Example 
JsNgram.keySeparator = '-';
Default value 
'/'
Usage

This is a constant string for the system. Users may change this to '-' for the flat index system.
