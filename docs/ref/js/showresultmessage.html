JsNgram.showResultMessage

.. Reference . JsNgram static class

method of static class JsNgram

Description

Show result message on error box.

Example 
JsNgram.showResultMessage([101, 17]);
// '101 hits in 17 documents.'
JsNgram.showResultMessage([0, 0]);
// 'Nothing found.'
Usage

showResultMessage(count);

Argument count is an array of two elements, hit count and document count. A text shown comes from two properties resultCount and resultNone.
