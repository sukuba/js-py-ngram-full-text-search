JsNgram.makeLinkToNext

.. Reference . JsNgram static class

method of static class JsNgram

Description

Generate html button to control pages for result.

Example 
selector.append(JsNgram.makeLinkToNext(true, selector, false, 200, 100));
Usage

makeLinkToNext(isPerfection, selector, doc, start, limit);

Argument isPerfection is a boolean value whether the button is for perfect matches or partial matches. Argument selector is a html element to place the next results. Argument doc is a boolean value whether the button is for next hits in a document, or next documents. Argument start is a number of items already shown. Argument limit is a number of items to show when the button pressed. A html div element of class ui icluding a button element with a click event showing next results is created.
