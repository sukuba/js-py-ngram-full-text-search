JsNgram.makeResultHtml

.. Reference . JsNgram static class

method of static class JsNgram

Description

Generate html for result. It has sub functions:

header
generate table headers.
content
generate table contents of hit.
title
generate table contents of doc info.
docbox
generate table contents of doc.
Example 
selector.append(JsNgram.makeResultHtml.header());
selector.append(JsNgram.makeResultHtml.content([x, pos+1]));
selector.append(JsNgram.makeResultHtml.title(url, title, []));
selector.append(JsNgram.makeResultHtml.docbox());
Usage

makeResultHtml.header(data);
makeResultHtml.content(data);
makeResultHtml.title(url, title, data);
makeResultHtml.docbox();

Argument data is an array of content each of that is put in span element. Argument url is a url for hyperlink. Argument title is used for the face text of hyperlink.

Override this method to customize the look of result.
