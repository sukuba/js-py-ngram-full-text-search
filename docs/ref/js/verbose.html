JsNgram.verbose

.. Reference . JsNgram static class

property of static class JsNgram

Description

The verbosity of debug messages.

Example 
JsNgram.verbose = 1;
Default value 
0
Usage

This is a constant number for the system. Users may change this to set the verbosity level of messages on console. Increasing the number will get more messages.
