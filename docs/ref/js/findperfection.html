JsNgram.findPerfection

.. Reference . JsNgram static class

method of static class JsNgram

Description

Pick up perfect match from sorted result of N-gram partial matches. Returns the perfect match.

Example 
var found = JsNgram.sortResultsByLocation(results);
JsNgram.findPerfection(found, 3);
Usage

findPerfection(x, n);

Argument x is a sorted result of match (by document). Argument n is the required count for perfect match.

This very method does the N-gram result matching.
