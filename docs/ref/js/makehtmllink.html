JsNgram.makeHtmlLink

.. Reference . JsNgram static class

method of static class JsNgram

Description

Generate a tag for document link.

Example 
JsNgram.makeHtmlLink('txt/a/b.txt', 'title of document B');
// <a href="txt/a/b.txt" target="_blank" 
//    type="text/plain; charset=utf-8">
// title of document B</a>
Usage

makeHtmlLink(url, title);

Argument url is a url to the document. Argument title is a document title to be shown. This uses the property linkAttributes to set attributes of a tag.
