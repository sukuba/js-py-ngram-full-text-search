JsNgram.generateDeferred

.. Reference . JsNgram static class

method of static class JsNgram

Description

Generate ajax request array for each N-gram keyword after submit.

Example 
JsNgram.work['deferred'] = JsNgram.generateDeferred(JsNgram.work);
// the last one submits multiple ajax requests.
Usage

generateDeferred(work);

Argument work is the work property.

Reads N-gram keywords from the property work.

It returns an array of jQuery deferred object that should be completed before reporting. Read the jQeury document for details.
